# 外部知识审查

外部知识审查检查论文、网页、GitHub 主题、草稿经验和综合判断是否可靠、可追溯、可分类、可复用。

它不直接决定外部世界真相；它记录证据质量、争议、时效性和知识库内的处理风险。

## 审查准则

- 来源是否可追溯。
- 结论是否和证据分开。
- 不稳定事实是否记录日期、版本或访问时间。
- 是否记录不确定性、冲突证据和方法限制。
- 需要分类的知识是否能落到三级领域。
- 是否避免大段复制受版权保护内容。
- 来源质量、样本、实验设计或实现限制是否可见。
- 条目是否能被索引、检索和项目工作流引用。

如果是内容质量问题，进入知识工作流；如果是可测试质量问题，进入 evals；如果暴露系统结构缺陷，进入 `diagnostic-system`。
