# Evidence-First Knowledge

## Definition

Evidence-first knowledge means every reusable judgment should remain connected to the source, context, and confidence level that support it.

## Why It Matters

This prevents the knowledge base from becoming a collection of attractive but ungrounded opinions. It also lets Codex distinguish stable principles from time-sensitive observations.

## Key Distinctions

- Source record: describes where information came from.
- Atomic note: describes a reusable judgment extracted from one or more sources.
- Synthesis: combines multiple notes and sources into a higher-level view.

## Evidence

- This is a repository operating principle rather than an external factual claim.
